Ajax Data Loading

Learn about this app

Make an ajax request via input field below.
Enter file location and press the "Load Data" button.
File location: "js/data/configurations.json" or "js/data/configurations_large.json"
Loading the "js/data/configurations_large.json" file will demonstrate lazy loading for handling large amounts of data.
Note: App created with foundation 6, js, jquery, motion ui, css, scss.
This app was created to learn basics of making ajax calls and then displaying the received json data.
